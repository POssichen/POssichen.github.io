毕业设计操作手册

2019-11-19 Views 毕业设计 2019 1 min read

虚拟环境搭建：

  1. 在终端中 cd 进入到文件所在的文件夹；
  2. 进入虚拟的 python3 环境中，运行的命令行为
    source ./env/bin/activate；
  3. 如果新设备还没有创建虚拟环境，可以运行命令行 python3 -m venv env 来创建虚拟环境；
  4. 可以使用 python3 -m pip install XXX 来安装所缺少的工具包；
  5. 遇到下载速度过慢就上网搜索国内的镜像库；
  6. 若没有安装/升级pip工具，需要先安装/升级pip工具
    python3 -m pip install pip
    python3 -m pip install --upgrade pip

资源包安装：

  1. python3 -m pip install utils
  2. python3 -m pip install numpy
  3. python3 -m pip install pandas
  4. python3 -m pip install sklearn
  5. python3 -m pip install keras
  6. python3 -m pip install tensorflow
  7. python3 -m pip install nltk
