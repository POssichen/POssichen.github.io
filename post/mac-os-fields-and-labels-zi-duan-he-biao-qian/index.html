Mac os Fields and Labels 字段和标签

2020-02-03 Views 2 min read

Combo Boxes 组合框

组合框在单个控件中结合了文本字段和下拉按钮。用户可以在字段中输入自定义值或者从预定义值中进行选择。用户输入自定义值时，不会将其添加到选择列表中。

  1. 使用列表中有意义的默认值填充该字段。
  2. 使用介绍性标签可以使用户知道预期的值。
  3. 根据用户的输入提供相关的选择。
  4. 限制用户只允许选择一个值，不允许多选。
  5. 确保文本字段不超出框位。

Labels 标签

标签时描述屏幕上的界面元素或提供简短信息的静态文本字段。

  1. 使标签简短易读。
  2. 引入控件时使用冒号。
  3. 使用系统提供的标签颜色来传达重要性。
  4. 使用户协议选择标签进行复制。

Search Fields 搜索字段

搜索框一般位于工具栏的最右方。

  1. 显示取消搜索按钮。
  2. 必要的时候显示搜索字段相关的提示。
  3. 在适当的时间去搜索，可以在输入内容后进行搜索也可以在用户点击Enter后进行搜索。
  4. 考虑隐私性。可能用户不希望看到所有搜索历史。
  5. 允许用户自行选择过滤的属性和规则。
  6. 搜索的属性中选择适当的按钮外观。

Text Fields 文字字段

文字栏时一个圆角矩形区域，用户可以在其中输入或编辑一行或者多行文本。

  1. 使用介绍性标签或占位符文本来帮助用户理解输入内容。
  2. 在用户输入的时候就开始验证准确性。
  3. 使用数字格式来辅助数字数据的输入。
  4. 相应地调整换行符。
  5. 敏感的文本使用安全的文本字段。
  6. 考虑使用拓展工具来显示不可见的文本完整信息。
  7. 尽可能使文本字段的大小与预期文本的数量匹配。
  8. 预留合适的文字间隔。

Token Fields 令牌字段

令牌字段是在文字栏中存在的多项选择器。
